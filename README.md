# FreeAPI: Product Listing Interface

## Description

This project is a modern, responsive product listing web interface built using React, Vite, and Tailwind CSS. It fetches dummy product data from the [FreeAPI Random Products endpoint](https://api.freeapi.app/api/v1/public/randomproducts) and displays it in a clean, structured e-commerce layout.

## Features

- **Data Fetching:** Seamlessly retrieves and handles product data from an external REST API.
- **Premium UI/UX:** Styled with Tailwind CSS v4, featuring glassmorphism elements, beautiful gradients, and engaging micro-animations on hover.
- **Responsive Grid:** The layout adapts flawlessly to different screen sizes, providing an optimal viewing experience on mobile, tablet, and desktop devices.
- **Loading States:** A skeleton loader component ensures users are engaged while data is being fetched.
- **Error Handling:** Graceful fallback UI in case of API failure or network issues.

## Technology Stack

- **React 19**
- **Vite** (Build Tool)
- **Tailwind CSS v4** (Styling)

## Getting Started

1. Navigate to the project directory:
   \`\`\`bash
   cd Product-Listing-Interface
   \`\`\`
2. Install the dependencies:
   \`\`\`bash
   npm install
   \`\`\`
3. Start the development server:
   \`\`\`bash
   npm run dev
   \`\`\`

Open your browser and visit `http://localhost:5173/` to view the application locally.
